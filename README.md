📊 Marketing & Lead Funnel Analysis Dashboard

📌 Project Overview

This project analyzes marketing and lead funnel data to understand how users move through different stages of the customer journey:

Visitors → Leads → Customers

The dashboard helps businesses identify:

- User drop-off points
- High-performing marketing channels
- Funnel conversion performance
- Areas requiring optimization

The project was developed using Python for data analysis and Matplotlib for dashboard visualization.

---

🎯 Objectives

- Analyze marketing funnel performance
- Calculate conversion rates
- Identify user drop-off stages
- Compare channel effectiveness
- Provide recommendations for improving conversions

📂 Dataset Information

The dataset contains marketing funnel data with the following fields:

Column| Description
Date| Campaign date
Channel| Marketing source/channel
Visitors| Website visitors
Leads| Generated leads
Customers| Converted customers

---

📈 Key Metrics

Visitor → Lead Conversion Rate

Measures how many visitors became leads.

Result: 17.84%

---

Lead → Customer Conversion Rate

Measures how many leads became customers.

Result: 23.04%

---

Overall Conversion Rate

Measures overall funnel efficiency.

Result: 4.11%

---

📊 Dashboard Components

The dashboard includes four analytical sections:

1. User Funnel Journey

Visualizes the flow:
Visitors → Leads → Customers

2. High-Quality Leads by Channel

Shows customer distribution across marketing channels.

3. Funnel Conversion Performance

Compares different conversion rates.

4. User Drop-off Analysis

Highlights stages with maximum user loss.

---

📌 Key Insights

- Total Visitors: 186,803
- Total Leads: 33,329
- Total Customers: 7,679
- Highest drop-off occurs at the Visitor → Lead stage.
- Overall conversion rate is relatively low, indicating optimization opportunities.
- Certain marketing channels generate higher-quality leads.
